# Pig Game

## Overview

Pig Game is a simple dice game where the players take turns rolling a single die, trying to reach a target score. The first player to reach or exceed the target score wins the game. This project is implemented using HTML, CSS, and JavaScript.

## How to Play

1. Open the `index.html` file in your web browser.
2. Enter the target score for the game.
3. Click the "Roll Dice" button to roll the dice.
4. The value of the rolled die will be added to your current score.
5. You can choose to roll again to increase your score or hold to keep your current score.
6. If you roll a 1, your current score resets to 0, and it's the next player's turn.
7. The first player to reach or exceed the target score wins the game.

## Features

- Responsive design: The game is designed to be playable on both desktop and mobile devices.
- Customizable target score: You can set the target score before starting the game.
- Dynamic player turns: The game keeps track of which player's turn it is and updates the UI accordingly.
- Animated dice rolling: Enjoy a visually appealing dice rolling animation.

## Technologies Used

- HTML: Used for structuring the webpage.
- CSS: Used for styling the webpage and making it responsive.
- JavaScript: Used for implementing game logic and interactivity.
